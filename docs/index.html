Type Constructors

All types in Fae can be created using the name of the type in between angle brackets. <Example>

This is called a "Type Constructor".

When a type constructor is used, a new instance of the type is created with all values set to their defaults.

There are five basic types in Fae, and their example type constructors along with their default values are shown in this example:

// Initializes a <Boolean> as false.
// Equivalent to let bool = false;

let bool = <Boolean>;


// Initializes a <Number> as 0.
// Equivalent to let num = 0;

let num = <Number>;


// Initializes an empty <String>.
// Equivalent to let str = "";

let str = <String>;


// Initializes an empty <Array>.
// Equivalent to let arr = [];

let arr = <Array>;


// Initializes an empty <Object> with no properties.
// Equivalent to let obj = {};

let obj = <Object>;
